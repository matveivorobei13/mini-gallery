мини галерея с постами

тут вы можете создавать свои посты которые никто не увидит
## Возможности

* просмотр постов
* просмотр конкретного поста
* создание постов
* поиск постов

## Технологии

* Node.js
* Express
* PostgreSQL
* multer
* zod


## Установка
npm init -y
nmp i express cors multer zod env

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
cd REPOSITORY
npm install
```

## Настройка

Создайте файл `.env`:

```env
DATABASE_URL=your_database_url
PORT=3000
лень дальше писать
```

## Запуск
node server.js
либо npx nodemon server.j с библиотекой nodemon
Режим разработки:

```bash
npm run dev
```

Продакшен:

```bash
npm start
```
